# 박물관 AI 비콘 시스템 프로토타입

이 프로젝트는 비콘 기술과 AI를 활용한 박물관 관리 시스템의 프론트엔드 프로토타입입니다.

## 주요 기능

### 방문객 앱
- 개인화된 전시 추천
- AI 도슨트 대화 인터페이스
- 맞춤형 관람 경로 제안
- 실시간 위치 기반 정보 제공

### 관리자 대시보드
- 실시간 방문객 모니터링
- 방문객 세그먼트 분석
- 비콘 배치 및 상태 관리
- AI 기반 예측 분석

## 기술 스택
- Next.js
- React
- Tailwind CSS
- Chart.js

## 설치 및 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 프로젝트 구조
- `/src/app/visitor` - 방문객 앱 관련 페이지
- `/src/app/admin` - 관리자 대시보드 관련 페이지
- `/src/components` - 재사용 가능한 컴포넌트
- `/src/lib/data` - 목업 데이터

## 라이센스
이 프로젝트는 비공개 프로젝트로, 무단 복제 및 배포를 금지합니다.
